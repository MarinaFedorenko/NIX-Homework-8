1. создаем мавен рут проект, состоящий из двух модулей: библиотека и мейн 
2. в мейне оздаем массив int[] ints = new int[100000] (тот случай, когда размер имеет значение, величина массива определяется вычислительными возможностями ваших ноутбуков)

        int[] ints = new int[100000];
        int index = 0;
        for (int i = 0 ; i < ints.length ; i++) {
            index = (int) Math.floor(Math.random() * 100000);
            ints[i] = index;
        }
        for (int anInt : ints) {
            System.out.println("anInt = " + anInt);
        }

3. в библиотеке пишем следующие util классы:

	 BubbleSort (Пузырьковая сортировка);
	 SelectionSort (Сортировка выбором);
	 InsertionSort (Сортировка вставками);
	 ShuttleSort (Челночная сортировка);
	 ShellSort (Сортировка Шелла);
	 MergeSort (Cортировка слиянием);
	 QuickSort (Быстрая сортировка);
	 
4. основная задача - залогировать время выполнения каждой из сортировок и сравнить с java Collection.sort() в наносекундах (System.nanoTime())
5. провести тестирование столько раз, чтобы можно было построить рейтинг сортировок по скорости
6. все логи пишем в файле sort.log
